# Pensionistas_Java 🏨
Este programa foi desenvolvido com base no "Curso mais didático e completo de Java e OO, UML, JDBC, JavaFX, Spring Boot, JPA, Hibernate, MySQL, MongoDB e muito mais" como parte de um exercício introdutorio a POO. 

Programa feito em Java, simula o funcionamento de uma pensão com 10 quartos. 

Primeiramente o programa irá pedir o número de quartos que serão locados. 

Após isso, ele pedirá os dados de cada pessoa para cada quarto, os dados solicitados são: nome, email e número do quarto. 

Após o preenchimento, o programa mostra os quartos ocupados, com a informação de cada inquilino. 

Tecnologias usadas: 
- Java
- Eclipse/SpringBoot
